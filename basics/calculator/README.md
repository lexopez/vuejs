# vue-project4

This template should help get you started developing with Vue 3 in Vite.

## Concept

state

- displayNumber - state for storing selected numbers and operation using interpolation and calculated that using val() method

logic

- appendToDisplay - function that append selected number and operation to calculator
- calculate - function that calculate the expression as string using val() method
- clearDisplay

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
