# vue-project7

This template should help get you started developing with Vue 3 in Vite.

## Concept

- rendering list
- conditionals in classes

state

- steps - state for storing the steps
- currentStep - state for storing current step

logic

- nextStep - function that increment the current step
- prevStep - function that decrement the current step

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
