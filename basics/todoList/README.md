# vue-project1

This template should help get you started developing with Vue 3 in Vite.

## Concepts

state
- newTask - state for storing new task that gonna be created
- tasks - state for storing arrays of tasks

logic
- addTask - function to add task
- removeTask - function to remove task

event
- @click
- @keyup.enter

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
