# vue-project2

This template should help get you started developing with Vue 3 in Vite.

## Concepts

- Data fetching
- Rendering data
- Adding loading and error state
- using try/catch
- toggling theme

States

- isDarkTheme - state for toggling theme
- searchQuery - state for storing search input
- searchResults - state for storing search result which is in arrays
- isLoading - state for determining the status of fetching
- error - state for errors returned from fetching data

Logic

- add function to query data
- add function to toggle theme
- add function for submitting request
- bind classes for dynamic style

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)
