# vue-project6

This template should help get you started developing with Vue 3 in Vite.

## Concept

state

- formdata - state that store data like name, email, and password in object form
- isNameValid - state to check name if valid
- isEmailValid - state to check email if valid
- isPassworValid - state to check password if valid
- isFormValid - state to check if the form properties pass the validation

logic

- submitForm - function that send request

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
