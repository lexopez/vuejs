# vue-project10

This template should help get you started developing with Vue 3 in Vite.

## Concept

event

- @input - event for listening input events or typing

state

- username - state to store search input for github username
- userProfile - state to store queried github user profile
- error - state to store errors during fetching data

logic

- getUserProfile - function that query github user profile

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
