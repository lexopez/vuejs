# vue-project5

This template should help get you started developing with Vue 3 in Vite.

## Concept

state

- items - state for storing items in arrays of objects

logic

- toggleAccordion - function to toggle the view of item's content

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
