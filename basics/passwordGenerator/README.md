# vue-project8

This template should help get you started developing with Vue 3 in Vite.

## Concept

state

- passwordLength - state to store password length
- includeUppercase - state that check for uppercase option
- includeNumbers - state that checks number option
- includeSymbols - state that check symbols option
- generatedPassword - state to store generated password from the function

logic

- generatePassword - function that generate password

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
